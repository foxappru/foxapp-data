# How to add your dictionary

1. Create a GitHub account: https://github.com/join  
2. Create a **public** repository named `foxapp-data`  
3. Add `.txt` files with this structure:

```
word: dictionary
dog: chien
cat: кошка
apple: manzana
```

Each line should have a **word**, a colon, a space, and its **translation**.

4. Go to https://foxapp.ru and type your GitHub username to search your dictionaries
